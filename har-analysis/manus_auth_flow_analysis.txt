==========================
1. Cloudflare Turnstile CAPTCHA: O cfCaptchaCode é gerado pelo widget JavaScript
   do Cloudflare Turnstile. Isso é extremamente difícil de automatizar sem um
   serviço de resolução de CAPTCHA (como 2captcha, anticaptcha, etc.)
   
2. Verificação de E-mail: O Manus envia um código de 6 dígitos para o e-mail
   que precisa ser inserido para completar o registro.

3. Código de Convite: O registro requer um código de convite válido (OB746IYA9QIKG)

CONCLUSÃO:
==========
Este HAR é do processo de registro/login no MANUS.IM, não do 2NR.
A automação é possível MAS requer:

- Código de convite válido
